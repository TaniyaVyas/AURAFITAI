# AuraFit AI ✨

AuraFit AI is an AI-powered fashion recommendation and shopping assistant built using Flask, Python, OpenCV, and Machine Learning.

## Features

- Upload selfie/image
- Skin tone analysis using Computer Vision
- Personalized outfit recommendation
- Smart shopping recommendations
- Cheapest product finder
- Similar product recommendation fallback
- Aesthetic fashion UI

## Tech Stack

- Python
- Flask
- OpenCV
- Scikit-learn
- Pandas
- HTML/CSS

## How to Run

```bash
pip install -r requirements.txt
python app.py
